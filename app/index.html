Graph legend

Each dot is a dictionary category. Dot diameter scales with how many parent/child link rows involve that category: from _pdbx_item_linked_group_list when present, otherwise from _item_linked relationships in item definitions (e.g. some extension dictionaries). Hue indicates the category’s primary PDBx group (first category_group that is not inclusive_group, so the umbrella group does not paint every node the same colour).
